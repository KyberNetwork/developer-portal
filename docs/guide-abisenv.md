The contract Application Binary Interface (ABI) is the standard way to interact with the smart contracts in Ethereum. For trading or getting rates, the main smart contract to interact with is the [KyberNetworkProxy](api-kybernetworkproxy.md) contract for both mainnet and in the different testnets. Provided below is the canonical ABIs.
