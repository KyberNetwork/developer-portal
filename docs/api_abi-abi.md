The contract Application Binary Interface (ABI) is the standard way to interact with the smart contracts in Ethereum. 

We recommend importing the interfaces for the following functionalities:
- [`IKyberNetworkProxy`](#ikybernetworkproxy): Fetch rates and execute trades
- [`ISimpleKyberProxy`](#isimplekyberproxy): Simple APIs for trade execution
- [`IKyberHint`](#ikyberhint): Building and parsing hints
- [`IKyberStorage`](#ikyberstorage): Get reserve IDs for building hints
- [`IKyberFeeHandler`](#ikyberfeehandler): Claim staker rewards, reserve rebates or platform fees
- [`IKyberReserve`](#ikyberreserve): Fetch rates of a specific reserve
- [`IERC20`](#ierc20): Token operations (Eg. token transfers / approvals)

The full contract ABIs are also given below the interface section.
